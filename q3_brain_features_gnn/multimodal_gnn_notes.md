# GNN / 多模态融合说明

## 特征提取

使用统一图谱，以 ROI 作为节点。

### 边

- fMRI（功能连接 FC）：提取各脑区时间序列，计算两两 Pearson 相关系数，并做 Min-Max 归一化。
- DTI（结构连接 SC）：用 iFOD2 生成流线，再用 SIFT2 根据截面积赋权。计算连接矩阵后，利用连接区域灰质体积标准化，最后保留每个节点权重最大的前若干条连接。

### 节点

- sMRI（解剖学统计特征 AS）：顶点数量、表面积、灰质体积、皮层厚度（均值和标准差）、曲率、折叠指数。
- 内在神经时间尺度（INT, 来自 fMRI）：量化神经信号在局部回路中存储的持续时间，反映功能层级。
- 结构-功能耦合（SC-FC Coupling）：计算每个脑区 SC 和 FC 向量之间的 Spearman 等级相关系数。

## 多模态融合

1. 初始图由 FC 或 SC 构成。第一层 MaskGNN 让拓扑结构进入模型，输出初步节点嵌入。
2. 将节点属性向量（AS + INT + 耦合系数）与第一层节点嵌入拼接。
3. 拼接后的特征矩阵进入下一层 MaskGNN 深度计算。

## 预测和解释

- 看“边”：学习边权重掩码矩阵，权重高的边对应重要脑网络连接通道。
- 看“点”：对拼接进来的解剖学特征用 Grad-RAM / Grad-CAM 回推贡献度。
