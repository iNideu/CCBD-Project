# Q1 - 不平等厌恶如何发展

Notion 当前分类：

- 行为表现
- 参数拟合（`alpha`, `beta`, `lambda`）
- 表型分类
- 表型发展

## Files

- `fit_fehr_schmidt_params.py` - Fehr-Schmidt 模型多起点参数拟合、伪 R2 过滤、清洗后数据导出。
- `plot_linear_trajectories.py` - `alpha` 和 `beta` 随年龄变化的线性发育轨迹图。
- `lpa_phenotypes.R` - 基于 `alpha`, `beta`, `lambda` 的 LPA 表型分类，以及年龄轨迹多项逻辑回归。
