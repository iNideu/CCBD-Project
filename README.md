# CCBD-Project

Code for Hulab CCBD Project, organized from the Notion page `脑计划v2 / V2` by current research-question categories.

## Structure

- `q1_inequity_development/` - Q1: 儿童青少年不平等厌恶如何发展？
  - 行为表现
  - Fehr-Schmidt 参数拟合 (`alpha`, `beta`, `lambda`)
  - 发育轨迹绘图
  - LPA 表型分类与表型发展
- `q2_environment_cognition/` - Q2: 环境与认知变量如何影响不平等厌恶发展？
  - 变量处理
  - 随机森林 / SHAP / 多项逻辑回归思路
- `q3_brain_features_gnn/` - Q3: 脑功能、脑结构特征如何影响不平等厌恶发展？
  - 灰质体积提取
  - DTI 结构连接矩阵生成
  - GNN / 多模态融合设计说明
- `q4_neural_basis_scca/` - Q4: 哪些脑特征发育构成环境与认知变量影响不平等厌恶的神经基础？
  - 稀疏典型相关分析（sCCA）
